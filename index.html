엄마 기분 예측기 만들기

엄마를 화나게 하지 않기 위해, 여러 행동을 입력으로 받아 결과를 예측하는 퍼셉트론 실습입니다. 각 행동은 했으면 1, 안 했으면 0으로 입력되고, 학생은 각 행동의 중요도(가중치)를 -5부터 5 사이에서 직접 정합니다.

1단계 중요도 정하기 2단계 사례 살펴보기 3단계 중요도 조정하기 4단계 새 사례 예측하기

1단계. 행동카드의 중요도(가중치) 정하기

각 행동이 엄마 기분에 얼마나 큰 영향을 주는지 점수를 정해 보세요. 좋지 않은 행동은 양수, 좋은 행동은 음수로 주면 됩니다.

입력 규칙

사례에서는 각 행동이 있으면 1, 없으면 0으로 입력됩니다.
퍼셉트론은 입력값 × 중요도를 모두 더해 결과를 판단합니다.

2단계. 엄마가 화났는지 화나지 않았는지 살펴보기

아래 사례에서 각 행동은 0 또는 1로 입력됩니다. 여러분이 정한 중요도로 계산했을 때 실제 결과와 어떻게 연결되는지 살펴보세요.

3단계. 가장 잘 맞추는 중요도 찾기

사례 결과를 더 잘 설명하도록 중요도를 다시 바꿔 보세요. 조정할수록 예측기의 성능이 달라집니다.

현재 성능 점검

고정된 기준점:

4단계. 새로운 사례 넣어서 예측하고 성능 점검하기

새로운 행동 조합을 보고 여러분의 예측기가 화남/안 화남을 얼마나 잘 예측하는지 확인해 보세요.
