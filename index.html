퍼셉트론 실습: 화남 예측기

여러 행동 신호를 입력으로 넣고, 결과를 관찰하며 규칙을 추리해 보세요. 어떤 행동은 화남 점수를 올리고, 어떤 행동은 점수를 낮출 수 있습니다. 처음에는 각 행동의 점수(가중치)가 숨겨져 있습니다. 여러 번 실험한 뒤 가중치를 공개하면, 퍼셉트론이 어떻게 판단하는지 확인할 수 있습니다.

1단계 입력 선택 2단계 결과 관찰 3단계 가중치 공개 4단계 새 사례 예측

1단계. 입력 노드 선택하기

아래 행동이 있었는지 골라 보세요. 각 행동은 퍼셉트론의 입력 노드입니다.

현재 선택

아직 선택한 행동이 없습니다.

2단계. 결과를 관찰하며 규칙 추리하기

실험 결과를 모아 보세요. 어떤 행동은 결과를 강하게 만들고, 어떤 행동은 화남 점수를 낮출 수도 있습니다. 아직 점수는 공개되지 않습니다.

아직 충분한 실험이 없습니다. 서로 다른 조합으로 여러 번 실험해 보세요.

3단계. 숨겨진 가중치와 역치 공개

이제 각 입력의 점수와 기준값을 확인해 보세요. 퍼셉트론은 입력값 × 가중치를 더한 뒤, 그 합이 역치를 넘는지 비교해 결과를 냅니다.

4단계. 새로운 사례 예측하기

이제 가중치와 역치를 바탕으로 새로운 입력 조합의 결과를 예측해 보세요.
